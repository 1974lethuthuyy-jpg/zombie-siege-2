# Zombie Siege 2D

Game bắn zombie 2D chạy trực tiếp bằng HTML, CSS và JavaScript thuần.

## File chính

- `index.html`
- `style.css`
- `game.js`

## Chạy local

Mở `index.html` bằng trình duyệt.

## Điều khiển

- `WASD`: di chuyển
- `Chuột`: ngắm
- `Giữ click trái`: bắn liên tục
- `1-4`: đổi vũ khí

## Đưa lên GitHub Pages

1. Tạo một repository mới trên GitHub.
2. Upload các file `index.html`, `style.css`, `game.js`, `README.md`.
3. Vào `Settings` -> `Pages`.
4. Trong mục `Build and deployment`, chọn:
   - `Source`: `Deploy from a branch`
   - `Branch`: `main`
   - `Folder`: `/ (root)`
5. Chờ GitHub publish, sau đó mở link dạng:
   - `https://your-username.github.io/your-repo-name/`

## Đưa lên Google Search

Sau khi site đã có link public:

1. Mở Google Search Console.
2. Thêm URL website.
3. Xác minh quyền sở hữu.
4. Gửi URL để Google index.

## Lưu ý

- Nếu sau này dùng domain riêng, nên thêm `sitemap.xml` và `robots.txt` theo đúng domain thật.
- Có thể nén ảnh và thêm favicon nếu muốn site trông hoàn chỉnh hơn.
